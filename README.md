## Описание программы
*"Random Task Generator — это графическое приложение для генерации случайных задач. 
Позволяет пользователям получать случайные задачи из предопределенного списка, фильт их по типам (учёба, спорт, работа) и сохранять историю выполненных задач в формате JSON. 
Кроме того, пользователи могут просматривать историю сгенерированных задач."*
---
## Требования для запуска
Чтобы запустить приложение, необходимо установить следующее:
- Python 3.6 или выше
- Стандартные библиотеки `tkinter`, `random`, `json`, `os` (входят в стандартную поставку Python)
