## Описание программы
*"Random Task Generator — это графическое приложение для генерации случайных задач.   
Позволяет пользователям получать случайные задачи из предопределенного списка, фильтровать их по типам (учёба, спорт, работа) и сохранять историю выполненных задач в формате JSON.   
Пользователи также могут просматривать историю сгенерированных задач."*
## Требования для запуска
Чтобы запустить приложение, необходимо установить:
- Python 3.6 или выше  
- стандартная библиотека `tkinter` (обычно входит в поставку Python для Windows и Linux; для macOS — иногда нужно установить отдельно)
**Дополнительных зависимостей не требуется.**
